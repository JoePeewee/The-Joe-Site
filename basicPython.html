What is Python and why should you learn it?

Python is an interpreted, object-oriented, high-level programming language. Originally released in 1991,
Python is known for being one of the most simple programming languages to learn due to the syntax
being comparitable to English. One of the biggest downside to Python is slower performance.

Getting started

Getting started, you will need an IDE, or integrated development environment. An example of
an IDE is Pycharm. In Python, the print function, print(), is used to output to the console.
you should also keep in mind that you do not need to declare the end of
a line, unlike some other programming languages.

Using variables

Variables are an important part of programming, in any language. Varibles store information
which can be accessed in later code. Varibles have diffrent types, such as strings, integers, and
booleans (true or false). Unlike most other programming languages, you do not need to declare
what type a variable is, as Python will figure it out by itself. To initialize a variable, just
put the name of the varialbe, an equal sighn, then the value you want to assign to the variable.
Examples:
variable = ‘string’
number = 1234
boolean = True

Using replace & join method

The replace function is used to find specific words inside a string and replace them with another word.
An example of this is:
yourVarible.replace(‘word you want to replace’, ‘word to replace with’)

The join function is to combine two or more strings and get one. Example:
yourVariable.join(firstString, secondString)
Another way to join strings is to use the ‘+’ symbol between the strings you want to join. Example:
newString = firstString + secondString

Using lists

List can contain any type of variable and as many as you want. To initialize a empty list, you write: myList = []. You can initialize
the list with elements by seperating them with a comma, myList = [firstElement, secondElement, thirdElement]
Lists are orded by index. The first element's index is 0, next is 1, etc. you can add elements to a list
by using the .append(nameOfElement) function and remove elements by using the .pop(elementIndex) function.

Using split method

The split method allows you to seperate a string into a list of mutiple strings where you want.
Example: myString.split(‘characterToSplitAt’)
If you leave the brackets empty, the default character to split at is spaces.

Using keyboard input

Keyboard inputs are exactly what they sound like, when someone types something into the console,
your code will read it and store it as a variable. To do this, you write yourVarible = input()
it's important to note that the input will always be a string, if you want it to be an integer, do this:
yourInteger = int(input())

if, else, and elif statements

If statments are useful to know and understand in coding. If statments allows the code to
output diffrent things depending on the input though logical conditions. Example of conditions include,
equals: a == b
not equals: a != b
less than: a < b
greater than: or a >; b
These are used in if statements like this:
if a == b:
you should also note that Python knows if code is inside an if statment if it is indented.
else statments are used after if statments, and will run the code only if the if statment is false,
so if a did not equal b in the if statment above. Example:
else:
elif statments are a combination of if and else statments. They are used instead of multiple if statements to
save resources and make the code run faster. If the if statment fails, the program will run the elif statment to
check if it is true, instead of checking two if statments everytime the program runs. Example:
elif a != b:

For and while loops

While loops loop using conditionals, which are mentioned in if, else, and elif statements to repeatly
run certain code as long as the condition is set to true. An example of this is:
varible = 1
while varible > 10:
varible += 1
The first line of this code inizilizes a varible, and sets the value to 'one'. The second line of the code
is the while loop. The while loop will loop the code inside of it as long as our varible is less than 10.
Inside the code, the varible is increased by 1 everytime the loop runs it. Once the varible is equal
to 10, the while loop will be false and stop looping the code.

For loops are similar to while loops, but the varible is inizilized in the for loops. Example of this:
for i in range (5, 11, 2):
In this line, the varible is i, which is set to 0 by Python. the for loop will run 4 times as i increases each time
the loop runs. The first number inside the range function is the startinig value of i, the second
is the max value of i before the loop stops, and the third value specifies how much i will increase by. The default
values for the first and third values is 0 and 1.

Using functions

Functions allow you to divide code into blocks that can be called on. If you have repeatitive
code that is used many times in your program, such as sometime like finding all the factors of a number,
it's more time efficent to use a function. Example:
def myFunction():
print(‘This is a function, don't forget to indent the code inside functions!’)
